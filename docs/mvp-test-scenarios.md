# VocaRush MVP 테스트 시나리오

## 1. 신규 사용자 온보딩

목적: 첫 사용자가 로그인 후 프로필을 만들고 홈으로 진입하는지 확인합니다.

1. 로그아웃 상태로 접속
2. Google 로그인 또는 이메일 로그인
3. 프로필이 없으면 온보딩 화면 표시
4. 이름 입력
5. 역할 선택: 학생 또는 선생님
6. 학습 과목 선택: EJU, JLPT, TOEIC, 기타
7. 현재 수준 선택 또는 직접 입력
8. 완료

기대 결과:

- `vocarush_profiles`에 `user_id`, `name`, `role`, `goal`, `current_level` 저장
- 홈 화면에 이름/과목/수준 표시
- 재로그인 후 온보딩이 다시 나오지 않음

## 2. 핵심 학습 흐름

목적: 세트 선택부터 학습 기록 저장까지 한 번에 작동하는지 확인합니다.

1. 홈에서 추천 세트 카드 선택
2. 세트 상세에서 `낱말카드` 또는 `학습하기` 선택
3. 3개 이상 문제 풀이
4. 일부 정답, 일부 오답 선택
5. 뒤로 가기
6. 홈 이동

기대 결과:

- 홈 `오늘 요약`의 풀이 수 증가
- 정답률 반영
- 오답이 있으면 최근 오답/약점 영역에 표시
- 로그인 사용자라면 `vocarush_learning_records`에 기록 생성

## 3. 재로그인 기록 유지

목적: Supabase 저장 기록이 다시 로그인해도 유지되는지 확인합니다.

1. 학습 기록 생성
2. 로그아웃
3. 같은 계정으로 다시 로그인
4. 홈 화면 확인

기대 결과:

- 기존 프로필 유지
- 최근 학습 기록과 정답률이 다시 표시
- 게스트 모드 기록은 서버에 저장되지 않음

## 4. 형광펜 단어장

목적: 사진/파일 업로드 UI와 추출 결과 저장 흐름을 확인합니다.

1. 라이브러리 > 개인/과제 > 형광펜 단어장 진입
2. `사진·파일 업로드` 선택
3. 이미지 파일 선택
4. 추출 결과 확인
5. 단어 1개 이상 선택
6. `선택 단어 추가`

기대 결과:

- Edge Function이 배포되어 있으면 OpenAI Vision 추출 결과 표시
- 실패하면 샘플 추출 결과 표시
- 선택 단어가 형광펜 단어장에 추가
- 복습 큐에 포함

## 5. 빈 상태 / 에러 상태

목적: 사용자가 막히지 않고 다음 행동을 이해하는지 확인합니다.

- 오답이 없는 상태에서 오답 복습 선택
- 형광펜 단어가 없는 상태에서 형광펜 단어장 진입
- Supabase 권한 오류 상태에서 프로필/학습 기록 저장 시도
- 네트워크 차단 후 학습 기록 저장 시도

기대 결과:

- 빈 상태 문구 표시
- 오류 원인을 한국어로 표시
- 다시 시도 또는 다른 행동 버튼 제공

## 6. 모바일 레이아웃

목적: 모바일 앱처럼 조작 가능한지 확인합니다.

뷰포트:

- 390 x 844
- 430 x 932
- 520 x 900

확인 항목:

- 하단 탭 터치 영역 충분
- 뒤로가기 위치가 왼쪽 상단
- 버튼 텍스트 잘림 없음
- 카드 폭이 과도하게 넓어지지 않음
- 입력창과 키보드 흐름이 자연스러움
